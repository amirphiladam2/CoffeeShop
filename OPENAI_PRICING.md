# OpenAI API Pricing & Free Tier

## Do You Have to Pay for OpenAI?

**Short answer**: Yes, but new accounts get **free credits** to start!

## Free Credits for New Accounts

When you create a new OpenAI account, you typically receive:
- **$5 in free credits** (as of 2024-2025)
- These credits are valid for ~3 months
- Perfect for testing and development!

With `gpt-3.5-turbo` at **$0.002 per 1K tokens**, $5 gives you:
- ~2.5 million tokens
- ~2,500 conversations (assuming 1,000 tokens per conversation)

## After Free Credits Run Out

1. **Pay-as-you-go**: Only pay for what you use
2. **No monthly fees**: Just pay for API calls
3. **Automatic billing**: OpenAI charges your card automatically
4. **Set spending limits**: Control your costs in OpenAI dashboard

## Setting Up Billing (Optional)

If you want to continue after free credits:

1. Go to https://platform.openai.com/account/billing
2. Add a payment method
3. Set spending limits if desired
4. Monitor usage in the dashboard

**You can set a monthly spending limit** (e.g., $10/month) to avoid surprises!

## Free Alternatives (If Needed)

If you want completely free options, consider:

### Option 1: Hugging Face (Free tier available)

### Option 2: Google Gemini (Free tier)
- Limited free requests
- Good quality
- Need to modify edge function

### Option 3: Local Models (Advanced)
- Run models on your own server
- Completely free but requires significant setup
